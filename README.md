## 🛠️ Tecnologías usadas

- ⚡ **Next.js (App Router)**
- 🧠 **Zustand** (State Management)
- 🎨 **Tailwind CSS**
- 🧩 **TypeScript**
- 🔔 **@pheralb/toast**
- 🎯 **Lucide Icons**

---

## 📦 Requisitos

Asegúrate de tener instalado:

- **Node.js** ≥ 18
- **npm** o **pnpm** o **yarn**

---

## ▶️ Cómo correr el proyecto

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/SamuelCB13/PruebaTecAcema.git
```

### 2️⃣ Instalar dependencias
```bash
npm install
```

### 3️⃣ Ejecutar en desarrollo
```bash
npm run dev
```

El proyecto estará disponible en:

👉 http://localhost:3000
